**IMRF** is a modern framework designed to transform static Markdown/MDX into modular, navigable, and highly customizable web documentation. It keeps GitHub as your source of truth while providing a rich, interactive reading experience.

## âœ¨ Key Features

- **mdx-remote Power**: Render Markdown/MDX on the server with full React component support.
- **Dynamic Routing**: File-system based routing that mirrors your `docs/` structure.
- **Auto-Navigation**:
  - Recursive **Sidebar** generated from file tree.
  - Automatic **Table of Contents** extracted from headings.
- **Custom Components**: Embedded alerts, cards, and interactive widgets directly in Markdown.
- **GitHub Integration**:
  - Dynamic **Status Badges** (like the ones above!) generated by the app.
  - Automatic **Open Graph Images** for every page.
- **Developer Tools**: Built-in `/debug` dashboard to validate links and metadata.
